Testing the System
  • Using JUnit
  • Testing from android


This Jnuit test is testing three main function of client
   (CheckDeliverablity,OrderID,OrderStatus) and the connection
   between client and server. The test is working on different return
   value from sever based on different opcode.

To test the Android application, one must add multiple foods of varying
quantities to test whether or not the final total works correctly.
In addition, this also tests if all ordered items show on the Order page.
Actually testing this functionality has thus far been successful. For example,
the addition of a $5.56 Burrito Supreme Combo shows up and has a price of
$5.66 ($.10 addition for drink) on the order page. Then, adding 2 more orders of
this food, shows up as having ordered 3 and the price is the correct value of $16.98.
