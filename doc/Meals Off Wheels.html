Meals Off Wheels
  • Server Application
  • Android Application
  • Testing with JUnit


An easy to use program to place and track taco bell orders via a drone delivery system.

The full system consists of an Android application that connects to a server which stores,
    returns, and manages the orders in a mysql database.

This docuementation serves to describe the usage, intent, and performance of the different
    parts of the application stack. Please use the links for more in-depth descriptions
    on each section.
