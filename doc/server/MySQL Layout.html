MySQL Database Layout

There are three tables in the database.  They all have different fields for their respective entries and are accessable by the server users. 
+----------------+
| Tables_in_mow  |
+----------------+
| orders         |
| stores         |
| users          |
+----------------+

The orders table. Contains fields for the unique orderID
    (used as the primary key), userID, the storeID,
    and delivery status information
+----------------+------------------+------+-----+---------------------+----------------+
| Field          | Type             | Null | Key | Default             | Extra          |
+----------------+------------------+------+-----+---------------------+----------------+
| ID             | int(10) unsigned | NO   | PRI | NULL                | auto_increment |
| UserID         | int(11)          | NO   |     | NULL                |                |
| StoreID        | int(11)          | NO   |     | NULL                |                |
| DeliveryLat    | double           | NO   |     | NULL                |                |
| DeliveryLng    | double           | NO   |     | NULL                |                |
| Approved       | tinyint(1)       | NO   |     | 0                   |                |
| TimeReceived   | timestamp        | NO   |     | CURRENT_TIMESTAMP   |                |
| TimeDeliveried | timestamp        | NO   |     | 0000-00-00 00:00:00 |                |
+----------------+------------------+------+-----+---------------------+----------------+



The stores table.  Contains fields for the unique storeID   
    (used as the primary key), the store's location and its status.
+------------+------------------+------+-----+---------+----------------+
| Field      | Type             | Null | Key | Default | Extra          |
+------------+------------------+------+-----+---------+----------------+
| ID         | int(10) unsigned | NO   | PRI | NULL    | auto_increment |
| StoreLat   | double           | NO   |     | NULL    |                |
| StoreLng   | double           | NO   |     | NULL    |                |
| Connection | varchar(50)      | NO   |     | NULL    |                |
| Open       | tinyint(1)       | NO   |     | 0       |                |
+------------+------------------+------+-----+---------+----------------+

The users table.  Contains fields for the unique userID   
    (used as the primary key), the user's name, contact info
    and default delivery location
+-------------+------------------+------+-----+---------+----------------+
| Field       | Type             | Null | Key | Default | Extra          |
+-------------+------------------+------+-----+---------+----------------+
| ID          | int(10) unsigned | NO   | PRI | NULL    | auto_increment |
| DeliveryLat | double           | NO   |     | NULL    |                |
| DeliveryLng | double           | NO   |     | NULL    |                |
| UserName    | varchar(50)      | NO   |     | NULL    |                |
| Email       | varchar(50)      | YES  |     | NULL    |                |
| PhoneNum    | varchar(16)      | YES  |     | NULL    |                |
| PreferEmail | tinyint(1)       | NO   |     | NULL    |                |
+-------------+------------------+------+-----+---------+----------------+
